# Tour Guide Web Application

## Description
The **Tour Guide Web Application** is a platform designed to help users explore and plan their trips to various tourist destinations. This web app allows users to browse tour packages, view detailed itineraries, and make bookings. With a simple and responsive design, it provides a smooth and enjoyable experience for travelers.

The app also includes a secure user authentication system, allowing users to sign up, sign in, and manage their accounts. Firebase is used for both authentication and storing user data and booking information.

## Features
- **User Authentication**: Secure signup and signin functionality with Firebase authentication.
- **Tour Packages**: Browse and explore a variety of tour packages, each with detailed descriptions, pricing, and itinerary information.
- **Booking System**: Users can select and book their desired tour packages.
- **User Profile**: Users can manage their profile and view their booking history.
- **Responsive Design**: A fully responsive design that works seamlessly on desktops, tablets, and mobile devices.
- **Real-Time Data**: Firebase is used for storing and retrieving real-time data, ensuring that the app is always up-to-date.
- **Interactive UI**: Smooth navigation with interactive elements for a better user experience.

## Technologies Used
- **Frontend**:
  - HTML
  - CSS
  - JavaScript
  - React.js
- **Backend**:
  - Firebase Authentication for user management
  - Firebase Firestore for real-time data storage
- **Development Environment**:
  - Vite for faster build and development
  - Firebase SDK for integrating Firebase services
